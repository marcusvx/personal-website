# marcusvinicius.info
A small personal project to keep my website online and try new things. Nothing too fancy.

[marcusvinicius.info](https://marcusvinicius.info)

## How to run

```bash
npm install
npm run dev
```   

## Tech Stack

- Astro — Static site generator
- React — Interactive components
- TypeScript — Typing
- Tailwind CSS — Styling
- Cloudflare Pages — Hosting
